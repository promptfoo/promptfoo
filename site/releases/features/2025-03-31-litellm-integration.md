# LiteLLM integration

Complete integration with LiteLLM's 100+ providers in a single interface:

```yaml

  - litellm:gpt-4.1
  - litellm:claude-3.7-sonnet
  - litellm:gemini-2.5-pro
  - litellm:llama-4-scout # New Llama 4 support
```

# Try LiteLLM integration
