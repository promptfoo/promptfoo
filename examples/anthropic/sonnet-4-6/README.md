# sonnet-4-6 (Claude Sonnet 4.6)

This example demonstrates Claude Sonnet 4.6's capabilities across reasoning, structured extraction, code generation, and summarization tasks.

You can run this example with:

```bash
npx promptfoo@latest init --example sonnet-4-6
```

## What This Tests

- **Reasoning**: Solving riddles and logic problems
- **Structured extraction**: Parsing unstructured text into JSON
- **Code generation**: Writing type-safe TypeScript with proper patterns
- **Summarization**: Producing concise, accurate summaries under constraints

## Running the Example

```bash
# Set your API key
export ANTHROPIC_API_KEY=your_api_key_here

# Run the evaluation
npx promptfoo@latest eval

# View results
npx promptfoo@latest view
```
