# empiriolabs (EmpirioLabs)

You can run this example with:

```bash
npx promptfoo@latest init --example empiriolabs
cd empiriolabs
```

## Usage

To get started, set your `EMPIRIOLABS_API_KEY` environment variable. You can create a key from the [EmpirioLabs dashboard](https://platform.empiriolabs.ai/dashboard/api-keys).

Next, edit promptfooconfig.yaml.

Then run:

```bash
promptfoo eval
```

Afterwards, you can view the results by running `promptfoo view`
