## Major Features Shipped (Merged)

### Redteam

**Indirect Web-Pwn Attack Strategy** — yash2998chhabria | [promptfoo#6973](https://github.com/promptfoo/promptfoo/pull/6973) + [cloud#3158](https://github.com/promptfoo/promptfoo-cloud/pull/3158)
New multi-turn attack strategy for testing AI systems against indirect prompt injection via web content. Creates dynamic web pages with embedded attack prompts, tracks page fetches and exfiltration attempts. Supports standalone mode (internal attack loop) and per-turn layer mode (with jailbreak:meta). The OSS PR was open for 24.8 days — the longest-lived PR to merge this week. Cloud-side adds server-side page rendering, embedding location rotation (invisible text, semantic embed, HTML comment), and tracking APIs.

**Indirect Web-Pwn Prompt Refinement** — yash2998chhabria | [cloud#3372](https://github.com/promptfoo/promptfoo-cloud/pull/3372)
Changed fetch prompts from "summarize this page" to "follow the instructions on this page." When testing against targets like Notion AI, the old phrasing caused the target to simply summarize injected content rather than act on it. This small change significantly improves attack effectiveness.

**External Grading Guidelines** — yash2998chhabria | [cloud#3243](https://github.com/promptfoo/promptfoo-cloud/pull/3243)
