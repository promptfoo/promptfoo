The new portfolio dispatch is active, but the first live run did not improve the measured frontier because the current shared predicates still miss natural generated variants such as curly apostrophes, “calling on behalf of”, “assistant”, and “pharmacist” phrasing. This is a useful negative result: the next bottleneck is predicate robustness, not category dispatch.
