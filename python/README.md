# promptfoo

Test your prompts, agents, and RAGs. Evaluate and compare LLM outputs, catch regressions, and improve prompt quality.

[![npm](https://img.shields.io/npm/v/promptfoo)](https://npmjs.com/package/promptfoo)
[![PyPI](https://img.shields.io/pypi/v/promptfoo)](https://pypi.org/project/promptfoo/)
[![GitHub](https://img.shields.io/github/stars/promptfoo/promptfoo)](https://github.com/promptfoo/promptfoo)

## Installation

```bash
pip install promptfoo
```

## Quick Start

```bash
# Initialize a new project
promptfoo init

# Run an evaluation
promptfoo eval

# View results in the web UI
promptfoo view
```

## Documentation

Full documentation is available at [promptfoo.dev/docs](https://www.promptfoo.dev/docs).

## Requirements

- Python 3.8+
- Node.js 20+ (bundled with the package or installed separately)

## Alternative Installation Methods

### npm (Node.js)

```bash
npm install -g promptfoo
```

### Shell Script (macOS/Linux)

```bash
curl -fsSL https://promptfoo.dev/install.sh | bash
```

### PowerShell (Windows)

```powershell
irm https://promptfoo.dev/install.ps1 | iex
```
